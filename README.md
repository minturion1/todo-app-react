# Todo App

Simple todo application built with React, TypeScript and Tailwind CSS.

## 🚀 Features

* Add new tasks
* Delete tasks
* Clean and responsive UI

## 🛠 Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite

## 📦 Installation

```bash
npm install
npm run dev
```

## 📸 Preview

Basic task management UI with add and delete functionality.

## 📌 Notes

This project was created as part of learning React fundamentals, state management and component structure.
